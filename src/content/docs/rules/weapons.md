## Ranged Weapon Changes

These changes focus on crossbows.

### Crossbow changes

- Proficiency bonus increased from `+2` to `+3`
- Damage increased from `1d8` to `1d10`
- Property changed from `load minor` to `load standard`
- Requires `STR 16` in order to load

### Superior Crossbow changes

- Proficiency bonus increased from `+2` to `+4`
- Damage increased from `1d8` to `1d12`
- Property changed from `load minor` to `load standard`
- Requires `STR 18` in order to load
